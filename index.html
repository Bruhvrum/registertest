Quick Start Tutorial

1. Download the game files using the link above

2. Unpack the archive to a location of your choice

3. Open settings.json and enter your registered username and password

4. Save the file

5. Launch alicia-launcher-cli.exe

6. Enjoy!

IMPORTANT WARNING: This is a temporary testing setup. Password and usernames are not hashed in this current version of the server! Even though your data will be secured on my server, I still highly recommend using temporary passwords or generally passwords you don't use at all nor care about.

Available Chat Commands

You can use these commands in the game chat (Ranch Chat) after logging in:

Give Item:

//give item <count> <item ID>

Easily generate this command using the helpful website: samuelll41.github.io/alicia/

Horse Parameters:

//horse parts <skinId> <facetd> <maneId> <tailId>

This command allows you to customize your horse's appearance and attributes.
