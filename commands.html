Available Chat Commands

For help generating commands for items and horses, please visit this helpful website: samuelll41.github.io/alicia/

Note: To ignore any parameter and keep its current value, simply specify 0 as the value.

Show Online Players

Displays the current player count and a list of all online players.

//online

Usage Example:

//online

Set Emblem

Sets your account's emblem. You must restart the game for the change to appear.

//emblem <ID>

Usage Example:

//emblem 26

Give Item

Gives your player an item. Claim it from the "Received Gifts" tab in the Gifts menu in the inventory.
Navigate like this: Inventory > Open Storage > Received Gifts

//give item <count> <tid>

Usage Example:

//give item 1 10050

Change Horse Coat & Parts

Changes the appearance of your horse's coat, face, mane, and tail. You must restart the game for the change to appear.

//horse parts <skinId> <faceId> <maneId> <tailId>

Usage Example:

//horse parts 14 0 5 5

Change Horse Proportions

Adjusts the physical proportions of your horse. The default value for each setting is 5. You must restart the game for the change to appear.

//horse appearance <scale> <legLength> <legVolume> <bodyLength> <bodyVolume>

Usage Example:

//horse appearance 10 10 10 10 10

Randomize Horse

Randomizes the appearance and parts of the horse you are currently riding. You must restart the game for the change to appear.

//horse randomize

Usage Example:

//horse randomize

Visit Player

Sets your next portal destination to another player's ranch. The player must be online. This is a one-time teleport; portal destinations will be random again afterwards.

//visit <player>

Usage Example:

//visit Chrommd

Set Character Model

Changes your character's model ID. Using 0 will make your character invisible.

//model <modelid>

Usage Example (Invisible Character):

//model 0

Help Command

Displays a list of the recently added available commands. Note that older commands are not included in this in-game list and can still be found on this webpage.

//help

Usage Example:

//help

Create New Character

Sends you back to the character creator screen to make a new character. Note that a client restart is required for the changes to take effect once the character is created.

//create

Usage Example:

//create

Give Preset

Gives you a variety of item presets for a quick give-all. Items will be given upon game restart. You can choose from these presets:

  • all - gives all of the items from the presets listed below
  • feed - all feed items
  • clean - all wash items
  • play - all play items
  • cure - all cure items
  • construct - all construction/housing-related items

You can also specify a number after the preset to give that amount of each item. If no number is specified, it defaults to 1.

//give preset <preset> <count>

Usage Example:

//give preset feed 100
