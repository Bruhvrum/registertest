Server Changelog: July 3rd

Hello everyone,

A small server update has been deployed. Please see the summary of changes below.

  • Randomized Horses: Upon character creation, you will now be assigned three random horses that you can switch between. You can also get a new random horse at any time by using the command //horse randomize in the ranch chat, which requires a client restart to take effect.
  • Character Voices: An issue has been resolved where character voices did not match the selected gender. This is now corrected.
  • Emblem Customization: A new feature has been implemented allowing emblem customization. You can change your emblem using the command //emblem <ID> in the ranch chat, followed by a client restart. A complete list of emblem IDs and a command generator can be found on SAMUELLL's website in the emblems section: samuelll41.github.io/alicia/
  • Automatic Restarts: The server is now configured to restart automatically following a crash. Please be aware that in the event of a crash, your data will most probably be lost, unless it was successfully saved prior to the incident.
  • General Fixes: This update also includes various background bug fixes and server stability improvements.

Important Notice Regarding Data Wipes

Please be advised that all player data has been wiped. This was a necessary step to accommodate ongoing development and recent server-side changes. As the project evolves, further data wipes will be required periodically to support future updates.

Disclaimer: This server is a test environment. As such, it is not stable and users may encounter crashes, bugs, and other unexpected issues.

Thank you for your understanding.
