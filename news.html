Server Changelog: September 5th

Hello everyone,

A new server patch has been deployed, incorporating the latest updates and stability improvements from the latest commits to the master branch. Please see the summary of changes below.

New & Updated Systems

  • Ranches Open by Default: Based on a recent community poll, ranches are now unlocked by default. You can still choose to lock your ranch if you wish.
  • Messenger Implementation (WIP): This is the first implementation of the friends list. It currently serves as a simple list of online ranches/players you can visit by clicking on an icon next to their name. It serves no other functionality as of right now, so adding friends, chatting, or sending emails will not work. It's a simple and better alternative to the //visit command.
  • Incubator Implementation (WIP): You can now incubate and hatch eggs! Because of this, you will no longer be able to give yourself a pet directly via the //give command, only eggs. Fear not, as the boost logic isn't fully implemented so you can boost your egg hatch infinitely until it's ready to hatch, and you won't get any refined crystals deducted as of right now either. So boost to your heart's desire! **NOTE OF A BUG!/incomplete feature:** Once you acquire the pet from the egg, summoning it will not summon the pet until you restart the client; however, if you wish to un-summon the pet, that will work without restarting. You can find the list of all of the eggs that you can give yourself as well as generate the give command on this website: https://samuelll41.github.io/alicia/.
  • New Item Presets Command: The //give command has a new sub-command "preset". You can choose from a variety of item presets for a quick give-all. Currently, you can choose from these presets:
    • feed: all feed items
    • clean: all wash items
    • play: all play items
    • cure: all cure items
    • construct: all construct/housing related items

    Example usage: //give preset feed

    Note: A client restart is required to see the items given by this command.

  • Horse Stamina Implementation: Horse stamina can now be refilled and will correctly deduct the amount of carrots from the player.
  • Character Information: Stats in the character info menu have been identified and are ready for future use, but for now, all are set to 0.

Guild Implementation Improvements

  • Initial Guild Ranking Implementation: An initial guild ranking system has been implemented for future use but currently does not have any functionality.
  • Leave Guild: You can now leave a guild. (The cooldown for joining/creating a new guild is not yet implemented.)
  • Guild Creation Fee: Creating a guild will now correctly deduct 3,000 carrots from the player and correctly update their current carrot amount.

Bug Fixes & Improvements

  • Horse Stats: Fixed incorrect assignment of some horse stats.
  • Housing: General improvements have been made to the housing implementation.
  • Storage: The storage system will now display the correct timestamp for when a gift has arrived, and a notification will pop up above the inventory.
  • General Improvements: General improvements to bug fixes have been implemented.

Notice: data has been wiped!

Please be advised that all player data has been wiped. This was a necessary step to accommodate ongoing development and recent server-side changes. As the project evolves, further data wipes will be required periodically to support future updates.

Thank you for playing and testing on my server! Please make sure to report any bugs and glitches you encounter so we can resolve them quickly. Thank you and have fun.

Disclaimer: This server is a test environment. As such, it is not stable and users may encounter crashes, bugs, and other unexpected issues.

Server Changelog: July 16th

Hello everyone,

Today marks a huge step forward for the server with the release of a brand new launcher and major quality-of-life updates. Please see the summary of changes below.

Launcher & Website Overhaul

  • New Story of Alicia Launcher: The old method of joining the server is now replaced! The new launcher makes joining easier than ever. Simply download the installer from the "Play" page, register an account, and click "Start Game". Your browser will prompt you to open the launcher, which will automatically log you in and connect you to the server. (OG Alicians might remember this "Web Launch" featureâ€”it's back and better than ever!)
  • Password Hashing & Auto-Conversion: The new launcher and backend system finally allow us to securely hash all passwords. For players registered before this update, your accounts have been automatically converted to the new format. You can log in with your old username and password right away!
  • Website Redesign: The website has been updated to support this new, streamlined login flow, providing a much easier player experience.

Server & Game Updates

  • More Ranch Animals: The client packaged with the new installer includes more ambient animals on the ranch, such as deers, eagles, and the cat!
  • Improved English Translations: The new client also includes updated translations, converting more of the original Korean text to English.
  • Ranch XP Fix: An issue with Ranch XP has been patched. All buildings and structures in the Ranch Management tab should now be buildable without being incorrectly level-locked.
  • Horse Armor Fix: A bug that caused horse armor to expire immediately or be un-equippable has been fixed.
  • General Improvements: This update also includes numerous other background bug fixes and server stability improvements.

Community Tools

  • Ranch Swapper Tool: For those interested in exploring, I've created a simple Python script to replace your default ranch with a race map or other special map. You can find it on GitHub here: SoA Ranch Swapper.

Data Integrity Notice

No data wipe was required for this update. All player data remains intact.

As always, thank you for playing and testing out the server. Please report any bugs you encounter!
